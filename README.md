# Portfolio Mariem Trabelsi

Portfolio React/Vite prêt pour un déploiement gratuit avec Vercel.

## Lancer en local

```bash
npm install
npm run dev
```

## Déployer sur Vercel

```bash
git init
git add .
git commit -m "portfolio initial"
git branch -M main
git remote add origin https://github.com/VOTRE-USERNAME/portfolio-mariem.git
git push -u origin main
```

Ensuite, importez le repository dans Vercel. Vercel détectera Vite automatiquement.

Commande de build : `npm run build`

Dossier de sortie : `dist`
